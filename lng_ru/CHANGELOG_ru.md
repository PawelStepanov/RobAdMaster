-------- version 0.0.01 ---------
Первая версия ПК:
- Добавлена возможность вводить параметры ОУ (1х1, 2х2, 3х3)
- Добавлен расчет возможной зоны неопределенности
- Добавлены расчеты регуляторов (номинального и робастного) для ОУ размерностью 2х2
- Добавлены настройки (Тмод, выбор критерия, ввод зоны неопределенности (общая для всех параметров))

-------- version 0.1.0 ---------
- Добавлен расчет робастно-адаптивного регулятора для ОУ размерностью 2х2
- Улучшена оптимизация перебора коэффициента для робастного регулятора

-------- version 0.2.0 ---------
- Переработаны базовые настройки (выбор критерия, ввод Тмод)
- Добавлены расширенные настройки (ввод зоны неопределенности для каждого параметра)

-------- version 0.3.0 ---------
- Увеличен функционал расширенных настроек (разное возмущение, перебор коэффициентов, перебирать номинальные или крайние значения)
- Добавлен excel-файл для сохранения расчетов

-------- version 0.4.0 ---------
- Добавлена возможность строить график переходного процесса 

-------- version 0.5.0 ---------
- Добавлен расчет номинального регулятора для ОУ размерностью 1х1 (Поисковый и беспоисковый метод)
- Добавлен расчет робастного регулятора для ОУ размерностью 1х1 (Поисковый, беспоисковый и метод МЧ)
- Добавлен расчет робастного регулятора методом МЧ для ОУ размерностью 2х2

-------- version 0.5.2 ---------
- Повторяющиеся функции вынесены в отдельные файлы

-------- version 0.6.0 ---------
- Добавлен расчет робастно-адаптивного регулятора для ОУ размерностью 1х1 (ПИ-регулятор)
- Добавлена возможность построить график переходного процесса для ОУ 1х1 с максимальной разницей
между робастным и робастно-адаптивным регулятором.

-------- version 0.6.1 ----------
- Исправлена ошибка, когда при запуске расчета робастного регулятора методом МЧ мог произойти сбой
- Исправлена ошибка, когда после расчета регулятора методом МЧ было сразу два звуковых уведомлений.

-------- version 0.7.0 ----------
- Оптимизирована аппроксимация объектов управления 
- Добавлена возможность рассчитывать параметры регулятора для ОУ размерностью 3х3

В будущих версиях планируется:
- Улучшить оптимизацию и ускорить вычисления регуляторов
- Доработать некоторые функции, чтобы уменьшить их количество
- добавить возможность выбора регулятора (И или ПИ) для ОУ размерностью 1х1
- Дать возможность вводить параметры номинального и робастного регулятора при расчете робастно-адаптивного регулятора
